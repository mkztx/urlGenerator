Url Generator

firstly input url you want to use in box Base/Your Url

Later you can add text which you want to add before your url

To do so, simply enter this text in "text before url" box

Generator have 2 modes:

Single and Mass mode

to use "single mode" enter how many rows you want to generate and press single mode

in single mode you have to enter every last text separately

To use Mass Mode you have to press "Mass Mode" button

In Mass Mode you can paste all your last texts in input box, texts are separated by enter, "." or ","

After input of your texts in both modes, you have to press generate to generate all urls

you can copy one by one, or press "Copy" button to display all urls in one text in lines
